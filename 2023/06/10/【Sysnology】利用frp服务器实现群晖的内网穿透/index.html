但是据测试 隔一段时间就断开了 再连接 得重启frpc客户端

这是成功的frps.ini配置

[common]
bind_port = 7000
vhost_http_port = 7001
dashboard_port = 7500
dashboard_user = admin
dashboard_pwd = admin

隔段时间断开的解决方案：设置心跳时间
frps.ini
heartbeat_timeout = 90//心跳值，负数禁用，不建议改动

frpc.ini
heartbeat_interval = 30//间隔时间
heartbeat_timeout = 90//超时时间
